## Codex Cloud란

**Codex Cloud** 는 로컬 머신이 아닌 OpenAI 클라우드 인프라에서 에이전트 작업을 실행하는 서비스입니다. 사용자가 요청을 보내면 클라우드 샌드박스에서 코드를 읽고 수정한 뒤, 결과를 Pull Request로 제출하거나 Slack 메시지로 알려줍니다.

주요 특징을 정리하면 다음과 같습니다.

| 특징 | 설명 |
|------|------|
| 백그라운드 실행 | 로컬 터미널을 닫아도 작업 계속 진행 |
| GitHub 통합 | PR 자동 생성, 코드 리뷰 코멘트 작성 |
| Slack 통합 | 작업 완료 알림, Slack에서 직접 작업 지시 |
| 자동 코드 리뷰 | PR에 대한 자동 리뷰 코멘트 생성 |

:::caution[ChatGPT 인증 필요]
Codex Cloud는 API 키가 아닌 **ChatGPT 계정 인증** 을 사용합니다. `codex auth login` 으로 ChatGPT 계정에 로그인해야 하며, API Key만으로는 사용할 수 없습니다.
또한 Codex Cloud에서는 **모델 선택이 불가** 합니다. OpenAI가 내부적으로 최적 모델을 자동 선택합니다.
:::

## codex cloud 명령

```bash
# 클라우드 작업 목록 보기
codex cloud list

# 새 클라우드 작업 생성
codex cloud "이 저장소의 README를 업데이트하고 PR을 열어줘"

# 특정 작업 상태 확인
codex cloud status <TASK_ID>

# 클라우드 작업 취소
codex cloud cancel <TASK_ID>
```

작업을 생성하면 `TASK_ID` (UUID 형태)가 반환됩니다. 이 ID로 상태를 조회하거나 결과를 로컬에 가져올 수 있습니다.

## codex apply — 결과를 로컬에 적용

클라우드에서 작업이 완료되면 변경사항을 로컬 워크스페이스에 직접 적용할 수 있습니다.

```bash
# 클라우드 작업 결과를 현재 디렉토리에 적용
codex apply <TASK_ID>

# 특정 디렉토리에 적용
codex apply <TASK_ID> --dir ./my-project
```

`codex apply` 는 클라우드가 만든 파일 변경사항을 로컬 git 워크트리에 패치 형태로 내려받아 적용합니다. 적용 전에 diff를 미리 확인하고 싶다면 `--dry-run` 플래그를 사용합니다.

```bash
codex apply <TASK_ID> --dry-run
```

<ConceptCard title="GitHub Actions와 Codex Cloud 연동">
GitHub Actions 워크플로에서 Codex Cloud 작업을 트리거할 수도 있습니다. 예를 들어 이슈가 생성될 때 자동으로 클라우드 작업을 시작하고, 완료되면 PR을 여는 파이프라인을 구성할 수 있습니다.

```yaml
# .github/workflows/codex-auto-review.yml
on:
  pull_request:
    types: [opened]
jobs:
  codex-review:
    runs-on: ubuntu-latest
    steps:
      - uses: openai/codex-action@v1
        with:
          task: "이 PR을 리뷰하고 개선 사항을 코멘트로 남겨줘"
          github-token: ${{ secrets.GITHUB_TOKEN }}
```
</ConceptCard>

## Claude Code와의 비교

<Compare>
  <Fragment slot="left">
  **Claude Code**
  - 로컬 실행 전용 (클라우드 백그라운드 작업 없음)
  - GitHub 연동은 사용자가 직접 커밋/PR 수행
  - Slack 연동 없음 (별도 MCP 서버로 구성 가능)
  - 모델을 자유롭게 선택 가능

  </Fragment>
  <Fragment slot="right">
  **Codex Cloud**
  - 클라우드 샌드박스에서 백그라운드 실행
  - GitHub PR 자동 생성 및 코드 리뷰 코멘트 내장
  - Slack 워크스페이스 연동으로 알림 수신
  - 모델 선택 불가 (OpenAI 자동 선택)
  - ChatGPT 인증 필수 (API Key 불가)

  </Fragment>
</Compare>

## 핵심 정리

- Codex Cloud는 **클라우드 샌드박스** 에서 백그라운드로 에이전트 작업을 실행합니다
- GitHub PR 자동 생성, 코드 리뷰 코멘트, Slack 알림을 기본 제공합니다
- `codex cloud "..."` 로 작업을 생성하고, `codex apply <TASK_ID>` 로 결과를 로컬에 적용합니다
- **ChatGPT 계정 인증** 이 필수이며, API Key 방식은 지원하지 않습니다
- **모델 선택이 불가** 하며 OpenAI가 내부적으로 최적 모델을 선택합니다
- Claude Code에는 직접 대응하는 클라우드 백그라운드 실행 기능이 없습니다
