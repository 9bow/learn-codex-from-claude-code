## 왜 별도 앱이 필요한가

터미널에서 `codex` 를 실행하면 하나의 에이전트와 대화합니다. IDE 확장도 마찬가지로 단일 에이전트 세션입니다. 작업이 하나일 때는 충분하지만, 여러 작업을 동시에 진행할 때는 한계가 있습니다.

예를 들어 다음 상황을 생각해 봅시다.

- 에이전트 A가 새 기능을 구현하는 동안
- 에이전트 B가 기존 테스트 스위트를 리팩터링하고
- 에이전트 C가 문서를 업데이트하고 있다

이 세 작업을 터미널 탭 세 개로 관리하면 각 작업의 상태를 파악하고, diff를 검토하고, 승인하는 과정이 흩어집니다. **Codex Desktop App** 은 이 모든 것을 하나의 인터페이스에서 관리하는 명령 센터 역할을 합니다.

## 주요 기능 개요

| 기능 | 설명 |
|------|------|
| **멀티 에이전트 관리** | 여러 에이전트를 동시에 실행하고 각각의 진행 상태를 한 화면에서 확인 |
| **워크트리 기본 지원** | 각 에이전트가 격리된 코드 복사본에서 작업, 로컬 git 영향 없음 |
| **diff 검토** | 시각적 diff 뷰어에서 코멘트 작성 후 에디터 즉시 열기 |
| **Automations** | 스케줄 기반 백그라운드 작업 설정 및 결과 검토 대기열 |
| **Skills UI** | Skills 생성·설치·관리 전용 인터페이스 |
| **CLI 세션 공유** | CLI에서 시작한 세션을 Desktop App에서 이어받기 가능 |

## 출시 정보

- **macOS**: 2026년 3월 초 출시
- **Windows**: 2026년 3월 4일 확장
- **Linux**: 2026-05 기준 미출시

:::note[참고]
Desktop App은 CLI와 같은 `~/.codex/config.toml` 을 공유합니다. CLI에서 설정한 모델, sandbox 정책, MCP 서버가 Desktop App에도 그대로 적용됩니다.
:::

## 요금제와 인증 (2026-05 기준)

Codex app은 ChatGPT 계정 또는 OpenAI API 키로 로그인할 수 있습니다. 다만 API 키로 로그인하면 ChatGPT credits 기반 기능이나 cloud threads 같은 일부 기능은 사용할 수 없습니다.

| 인증/플랜 | 사용 범위 | 특이사항 |
|------|---------|---------|
| **ChatGPT Plus / Pro / Business / Edu / Enterprise** | Codex app, CLI, IDE, web/cloud 기능 | 플랜별 사용 한도와 credits는 공식 Pricing 문서 기준 |
| **API key** | Codex app, CLI, SDK, IDE extension | Cloud 기반 기능(GitHub code review, Slack 등)은 미지원 |

:::note[참고]
사용량, credits, 프로모션은 수시로 바뀔 수 있습니다. 최신 한도는 [Codex Pricing](https://developers.openai.com/codex/pricing)과 Codex usage dashboard에서 확인하세요.
:::

<ConceptCard title="API 키 vs ChatGPT 계정">

Codex CLI와 IDE 확장은 `OPENAI_API_KEY` 환경 변수로 API 키 인증을 지원합니다. pay-per-token 방식으로 사용량만큼 과금됩니다.

단, 다음 기능은 ChatGPT 계정 기반 사용이 필요합니다.

- `gpt-5.5` 모델 접근
- GPT-5.3-Codex-Spark (Pro 전용)
- Codex Cloud 통합

CLI만 사용하는 경우 API 키로도 충분하지만, Desktop App을 활용하려면 ChatGPT 플랜이 필요합니다.

</ConceptCard>

## 핵심 정리

- Codex Desktop App은 **멀티 에이전트 명령 센터** 로, 단일 에이전트 도구인 IDE 확장·터미널과 역할이 다릅니다
- **macOS와 Windows** 를 모두 지원합니다 (2026-05 기준 Linux 미지원)
- CLI와 **`~/.codex/config.toml` 을 공유** 하므로 별도 설정 없이 바로 사용할 수 있습니다
- API 키로도 app에 로그인할 수 있지만, cloud 기반 기능은 ChatGPT 인증이 필요합니다
- 플랜별 사용 한도와 프로모션은 공식 Pricing 문서에서 확인해야 합니다
