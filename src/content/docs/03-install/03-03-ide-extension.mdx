Claude Code의 VS Code 확장(`anthropic.claude-code`)에 익숙하다면 Codex IDE 확장도 비슷한 흐름으로 설치할 수 있습니다. 가장 중요한 차이는 **CLI와 IDE 확장이 동일한 `~/.codex/config.toml`을 공유** 한다는 점입니다.

## VS Code 계열 확장 설치

VS Code Marketplace에서 Codex 확장을 설치합니다. 공식 확장 ID는 `openai.chatgpt` 입니다.

```bash
# CLI로 설치하는 방법
code --install-extension openai.chatgpt
```

또는 VS Code 내 확장 패널(`Ctrl+Shift+X` / `Cmd+Shift+X`)에서 `openai.chatgpt`를 검색해 설치합니다. Cursor와 Windsurf도 공식 설치 링크를 제공합니다.

설치 후 사이드바에 Codex 아이콘이 나타납니다. 처음 실행 시 인증 흐름(ChatGPT 또는 API 키)을 안내합니다.

## JetBrains 플러그인 설치

IntelliJ IDEA, PyCharm, WebStorm 등 JetBrains IDE에서:

1. `Settings` → `Plugins` → `Marketplace` 탭
2. "Codex" 또는 "OpenAI Codex" 검색
3. `Install` 클릭 후 IDE 재시작

:::note[참고]
JetBrains IDE 통합은 ChatGPT, API 키, JetBrains AI 구독 로그인을 지원합니다. 설치 링크와 지원 범위는 [공식 IDE 문서](https://developers.openai.com/codex/ide#jetbrains-ide-integration)를 기준으로 확인하세요.
:::

## config.toml 공유 구조

<Compare>
  <Fragment slot="left">
  - `~/.claude/settings.json`: Claude Code 전용
  - VS Code 확장 설정은 VS Code settings.json에 별도 저장
  - CLI와 IDE 설정이 분리됨

  </Fragment>
  <Fragment slot="right">
  - `~/.codex/config.toml`: CLI + IDE + Desktop App 공유
  - 단일 설정 파일로 모든 환경 관리
  - 프로젝트별 `.codex/config.toml`도 공유

  </Fragment>
</Compare>

설정 파일 위치는 환경 변수 `CODEX_HOME`으로 변경할 수 있습니다.

```toml
# ~/.codex/config.toml 예시
model = "gpt-5.4"
model_reasoning_effort = "medium"
```

## 모델 선택기 (Model Selector)

IDE 확장 내에서 `/model` 명령 또는 UI의 모델 선택기 드롭다운으로 현재 세션의 모델을 변경할 수 있습니다.

:::caution[Profiles 미지원]
config.toml의 `[profiles.<name>]` 기능은 현재 **IDE 확장에서 지원되지 않습니다** (Experimental, CLI 전용). 프로필별로 다른 모델을 사용하려면 CLI에서 `codex --profile <name>`을 사용하세요.
:::

## file_opener 설정

Codex가 파일을 열 때 사용할 에디터를 config.toml에 지정할 수 있습니다.

```toml
# VS Code로 파일 열기
file_opener = "vscode"

# 또는 Cursor
file_opener = "cursor"
```

지원하는 값: `"vscode"`, `"vscode-insiders"`, `"windsurf"`, `"cursor"`, `"none"` 입니다. 기본값은 `"vscode"` 입니다.

<ConceptCard title="Claude Code와 설정 동기화">
Claude Code는 VS Code 확장과 CLI가 별도의 설정을 가집니다. Codex는 단일 config.toml로 통합되어 있어, CLI에서 모델을 변경하면 다음번 IDE 확장 세션에도 자동으로 반영됩니다. 단, 세션 중 `/model` 명령으로 변경한 값은 해당 세션에만 적용됩니다.
</ConceptCard>

## 핵심 정리

- VS Code 계열 확장의 공식 ID는 `openai.chatgpt` 입니다
- JetBrains IDE 통합은 공식 IDE 문서의 JetBrains 설치 링크를 따릅니다
- CLI, IDE 확장, Desktop App이 `~/.codex/config.toml`을 공유합니다
- 모델 선택기(`/model`)로 세션 중 모델을 변경할 수 있습니다
- `file_opener` 설정으로 파일을 열 에디터를 지정할 수 있습니다
- `[profiles]` 기능은 IDE 확장에서 지원되지 않으므로 CLI에서만 사용 가능합니다
