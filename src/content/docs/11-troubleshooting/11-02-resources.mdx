## 핵심 정리

- Codex 공식 문서는 `developers.openai.com/codex/` 아래에 기능별로 정리되어 있습니다
- 프롬프트 품질을 개선하려면 Prompting, Best practices, Workflows, Codex Prompting Guide를 함께 확인하세요
- CLI 소스 코드와 이슈 트래커는 `github.com/openai/codex` 에 있습니다
- 공식 스킬 카탈로그는 `github.com/openai/skills` 에서 확인할 수 있습니다
- Agent Skills 표준 사양은 `agentskills.io/specification` 에서 확인할 수 있습니다
- 커뮤니티 질문은 GitHub Discussions 또는 OpenAI Community Forum을 활용하세요
- Claude Code 문서는 `docs.claude.com` 과 `docs.anthropic.com` 에서 확인할 수 있습니다
